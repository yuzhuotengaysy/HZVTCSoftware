1008: 方块国

题目描述

多年来，计算机科学家都在为一些复杂的计算进行着研究。已经有很多可行的算法可用，比如一些简单的问题，像排序、图中找到最短路等。比如邮差问题，就是经典的计算机算法问题。给定N个
城市，城市之间有路相连，请你计算出邮差从一个点出发，一趟下来经过所有的城市并回到起点的最短路。
方块国的总统雇佣你来设计一个程序计算一次经过所有城市的邮差路径。在方块国，每个城市都在矩形的顶点上。城市之间的道路沿着着北、西北、西、西南、南、东南、东、东北方向（如果该方
向有另一个相邻城市的话），城市和城市之间的正南正北距离、正东正西距离都是1个单位长度，所有的道路都是直线，如图所示2*3的方块国，邮差走一趟经过所有城市的最短路径为6个单位长度

输入

第一行输入方块国的数量t。
接下来t行，每行描述一个方块国，给定两个整数m和n，分别表示这个方块国的规格。满足1 < m < 20000，1 < n < 20000

输出

每个方块国输出两行，第一行"Scenario #i:"，i表示第几个方块国，从1开始计数。下一行，输出邮差最短路径，小数点后面保留2位数字。
每组结果之间输出一行空行。

样例输入

2
2 2
2 3

样例输出

Scenario #1:
4.00

Scenario #2:
6.00

题意

求最短路径

C++解法

#include<iostream>
using namespace std;

int main()
{
    int T;
    int m,n;
    cin>>T;
    cout << fixed << setprecision(2);
    for(int i=1; i<=T; ++i)
    {
        cin>>m>>n;
        cout<<"Scenario #"<<i<<":\n"; 
        if((m&1) && (n&1))  cout<<(m*n+0.41);
        else cout<<((double)m*n);
        cout<<endl<<endl;
    }
    return 0;
}
