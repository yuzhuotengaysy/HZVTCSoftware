1006: 判闰年

题目描述

老杨一到二月就犯糊涂，这个二月到底有几天？

输入

输入有三个数，分别表示三个年份

输出

输出这三个年份的二月份有几天，每个数据占一行

样例输入

2012
2013
2014

样例输出

29
28
28

题意

判断闰年

JAVA解法

还没写,坐等大佬!

C++解法

#include<iostream>
using namespace std;

int main() {
    int N=3;
    while (N--) {
        int year;
        cin >> year;
        if (year % 4 == 0 && year % 100 != 0 || year % 400 == 0) cout << 29 << endl;
        else cout << 28 << endl;
    }
    return 0;
}
