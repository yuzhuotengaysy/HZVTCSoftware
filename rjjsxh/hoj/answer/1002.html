1002: A+B问题

题目描述

给你两个整数，计算他们的和

输入

两个整数a,b (0<=a,b<=10)

输出

输出a+b的值

样例输入

1 2
3 5

样例输出

3
8

题意

单纯的A+B问题.

JAVA解法

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        while (in.hasNextInt()) {
            int a = in.nextInt();
            int b = in.nextInt();
            System.out.println(a + b);
        }
    }
}

C++解法

#include <iostream>
using namespace std;

int main()
{
    int a,b;
    while(cin >> a >> b)
        cout << a+b << endl;
}
